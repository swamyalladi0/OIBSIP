How To Run (Step by Step)
Step 1:

Install Python (if not installed)

Step 2:

Open terminal inside project folder

Step 3:

Run server first:

python server.py

Step 4:

Open new terminal and run client:

python client.py

Step 5:

Open another terminal and run another client to test multi-user chat.
